# CAP 6 — 1.0.0

Application Android marine complète : GPS, distance au littoral, repère 6 NM, ports, routage, suivi de route, trace GPX, profil bateau, pêche, guide de balisage et VHF/SOS.

## Important

CAP 6 est une aide à la navigation. Les jeux de données nautiques inclus dans cette première livraison complète restent partiels ou indicatifs lorsqu'ils ne proviennent pas encore d'une source officielle vérifiée. L'application signale cette limite et ne doit pas être utilisée comme unique moyen de navigation.
